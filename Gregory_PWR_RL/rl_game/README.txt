Robot Sample Game

1. You will need to install open cv in python
    pip install opencv-python

2. You will need to install a virtual machine called Pillow (imported as PIL)
    pip install pillow

The "player" is in blue
The "food" is in green
The "enemy" is in red

Currently it runs 25,000 episodes and the opencv shows the env every 1000 episodes
